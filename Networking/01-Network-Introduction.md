# What Is A Network?
- Set of interconnected computers sharing resources (images, files, videos, etc.) located on or provided by network nodes (hosts).
## Hosts
- Any device connected to a network, such as:
  - Laptop
  - Phone
  - Server – Provides services or resources to other devices (clients) on the network.
  - Router
  - Switch
### The world's largest network of interconnected networks is INTERNET.
## IP Address
- Logical address that uniquely identifies a host on the network. 
- Two versions available: IPv4 and IPv6
- Ways of configuring IP addresses: 
  - Static (manually)
  - Dynamic (DHCP: Dynamic Host Configuration Protocol)
## DNS
- Stands for Domain Name System
- Converts a domain name into its corresponding IP address so that devices can locate the requested resource.
## Port Number
- Identifies a specific service or application running on a device (e.g., HTTP: 80, HTTPS: 443).
## Communication Protocols
- Set of rules used by devices to communicate with each other. You can think of it as a language.
## NIC
- Stands for Network Interface Card
- Hardware component that allows a device to connect to a network.
